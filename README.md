# 🎓 English Flashcards

İngilizce kelimeleri öğrenmek için modern, eğlenceli ve çocuk dostu bir web uygulaması.

## ✨ Özellikler

- 📥 **CSV Yükleme**: Noktalı virgül ayrılmış CSV dosyasından kelimeler yükle
- 🎴 **3D Flip Animasyonu**: Kartları döndürerek çevirisi gör
- 🔊 **Sesli Telaffuz**: Hoparlör butonuyla kelimeyi dinle
- 📊 **İlerleme Çubuğu**: Öğrenme ilerlemesini izle
- 🎉 **Konfeti Efekti**: Tüm kelimeleri öğrenince kutlama
- ⌨️ **Klavye Desteği**: Ok tuşları ve boşluk ile navigasyon
- 🎲 **Shuffle**: Kartları rastgele sırayla karıştır
- 🎨 **Çocuk Dostu Tasarım**: Pastel renkler ve eğlenceli arayüz

## 🛠️ Teknolojiler

- **React**: UI kütüphanesi
- **Tailwind CSS**: Styling
- **PapaParse**: CSV parsing
- **React Confetti**: Konfeti efekti
- **React Icons**: İkonlar
- **Web Speech API**: Telaffuz

## 🚀 Başlangıç

### Gereksinimler
- Node.js 14+ ve npm

### Kurulum

```bash
# Bağımlılıkları yükle
npm install

# Uygulamayı çalıştır
npm start

# Tarayıcı otomatik açılacak: http://localhost:3000
