# English Flashcards 📚

Spaced Repetition System (SRS) ile İngilizce kelimeleri etkili bir şekilde öğrenin.

## Özellikler

✨ **Spaced Repetition Algorithm** - SuperMemo-2 benzeri SRS sistemi
💾 **Local Storage** - İlerlemenizi otomatik kaydeder
📊 **İlerleme Takibi** - Dinamik ilerleme çubuğu ve kelime sayacı
⌨️ **Klavye Kısayolları** - Hızlı navigasyon
🎯 **Akıllı Tekrar** - Yalnızca tekrar vakti gelmiş kelimeleri gösterir

## Kullanım

1. CSV dosyası yükle (Ayırıcı: semicolon `;`)
2. "Know" veya "Don't Know" tıkla
3. İlerleme otomatik kaydedilir

## Kurulum

```bash
npm install
npm start
```

## Klavye Kısayolları

- **Sağ Ok** → Bildim (Know)
- **Sol Ok** → Bilemedim (Don't Know)
- **Aşağı Ok** → Sonraki (Next)
- **Yukarı Ok** → Önceki (Previous)
